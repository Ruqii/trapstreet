# Brand Assets

Canonical brand specification artifacts for Trap Street.

## Files

| File | Purpose |
|---|---|
| `brand-sheet-v1.png` | The locked brand system reference sheet (v1.0). All visual decisions on this sheet are codified in [`../../docs/brand-system.md`](../../docs/brand-system.md). |

## Versioning

Brand sheets are append-only. When a new version locks, save it as
`brand-sheet-v2.png` and update `docs/brand-system.md` decision log. Never
overwrite `v1.png`.

## Note on file format

`brand-sheet-v1.png` is technically JPEG content with a `.png` extension.
This works in all browsers and git. Re-export as a true PNG (transparent
background) when convenient.
