# Error State Assets

Pixel-art illustrations used for non-success states across `trapstreet.run`.

The brand voice for errors is **"empathetic absurdity"** — no frowny corporate
"oops! something went wrong" copy. The manta has feelings, the page has feelings,
the user is in good company.

---

## Files

### `manta-stuck-404.png` — 404 Not Found

A sad pixel manta ray trapped inside a clear glass of water, looking forlornly
out at the viewer. Bubbles around it. Light blue palette, matches the primary
mascot.

**Visual story:** *"This page is like our manta in this glass — somewhere it
doesn't belong."*

**Used on:**
- `/404` — page not found
- Empty-state for "tool not found" lookups
- Any "this submission doesn't exist" state

**Suggested copy variants:**

```
404 — Off the map
The street you're looking for isn't on any of our maps.
(Trap streets are like that.)

[← Back to the leaderboard]
```

```
404 — Stuck
Our manta got stuck in a place it doesn't belong.
You probably did too.

[← Back to the leaderboard]
```

| `manta-empty-leaderboard.png` | empty state | Manta alone in an empty arena |
| `manta-rate-limit.png` | 429 | Manta with a tiny stop sign |

These keep the brand mascot present in moments where users are typically frustrated.
Each illustration humanizes the failure, lengthens the brand surface area, and
gives marketing free shareable content.

## File-naming convention

```
manta-<emotion-or-pose>-<purpose>.png
```

Examples: `manta-stuck-404.png`, `manta-asleep-503.png`,
`manta-confused-422.png`.

Source PSD/Aseprite files (when available) live alongside with the suffix
`-source.psd` or `-source.ase`.
